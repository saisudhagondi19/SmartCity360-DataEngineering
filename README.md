# 🏙️ SmartCity360 — Data Engineering Project

End-to-end Data Engineering capstone project built using 
Azure, Databricks, PySpark, Delta Lake and Power BI.

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Ingestion | Azure Event Hubs, Azure Data Factory |
| Storage | ADLS Gen2, Delta Lake |
| Processing | Databricks, PySpark |
| Analytics | Synapse Analytics, SQL Views |
| Visualization | Power BI |
| Orchestration | ADF Pipelines, GitHub Actions |

---

## 📁 Project Phases

- ✅ Phase 1 — Architecture Design
- 🔄 Phase 2 — Data Ingestion (APIs)
- ⏳ Phase 3 — Event Hubs Streaming
- ⏳ Phase 4 — Bronze Layer (Delta Lake)
- ⏳ Phase 5 — Silver Layer (PySpark)
- ⏳ Phase 6 — Gold Layer (SQL)
- ⏳ Phase 7 — ML with MLflow
- ⏳ Phase 8 — Orchestration (ADF)
- ⏳ Phase 9 — Power BI Dashboard
- ⏳ Phase 10 — DevOps & CI/CD
